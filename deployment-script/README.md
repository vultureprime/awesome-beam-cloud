## Features
- ตรวจสอบความถูกต้องของ GPU (Driver, Docker Runtime) 
- Nvidia Driver
- Nvidia Docker Runtime
- LLMs Dependency & Library 

## How to use
- Add execute permission
```
chmod +x deploy.sh
```

- Execute script
```
sudo ./deploy.sh
```
